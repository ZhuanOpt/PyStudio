# PyStudio

#### 介绍
PyStudio（暂定名称，曾用名pandaxstudio），是基于C# WPF的Python数据处理环境。

#### 软件架构
基于VS2019 .net4.7


#### 安装教程

1. 安装VS2019，配置开发环境为 .NET桌面开发
2. 安装Python（完整版PyStudio的后续版本已计划集成Python，请期待！）



#### 参与贡献
